1) Where to get the missing data from (if missing) (for every subject):
Latest Visit - filled from SV tab
Subject Status - filled from SV tab

Missing Visits - present in Visit_Projection_Tracker
Missing Page - present in Missing_Pages_Report

Coded Terms - count from GlobalCodingReport_WHODD + GlobalCodingReport_MedDRA
Uncoded Terms - count from GlobalCodingReport_WHODD + GlobalCodingReport_MedDRA
Open issues in LNR - count from Missing_Lab_Name_and_Missing_Ranges
Open Issues reported for 3rd party reconciliation in EDRR - count from Compiled_EDRR

Inactivated forms and folders - present in Inactivated_Forms_Folders_Records_Report

eSAE dashboard review for DM - count from eSAE_Dashboard_Standard_DM_Safety_Report
eSAE dashboard review for safety - count from eSAE_Dashboard_Standard_DM_Safety_Report

Total CRFs with queries & Non-Conformant data -
	1) Forms in Non conformant tab 
	2) Forms in Query Report - Cumulative
	Total count = Forms present in both (1) and (2)
	
Total CRFs without queries & Non-Conformant data -	
	1) Forms in Non conformant tab 
	2) Forms in Query Report - Cumulative
	Total count = Forms present in both (1) but not in (2)
	
Shortcut -> (Total CRFs with queries & Non-Conformant data) + (Total CRFs without queries & Non-Conformant data) = Pages with Non-Conformant data

% Clean Entered CRF = (Pages Entered - Pages with Non-Conformant data)*100 / (Pages Entered)

Previous Visits of subject present in SV tab

2) Data Validation and Data Mapping :

Expected Visits - count of visits from SV tab which does not have Screening or Unplanned Visit as Folder Name + count of visits from Visit_Projection_Tracker
Pages with Non-Conformant data - Page count from Non conformant tab

DM Queries - Queries under "DM Review" OR "DM from System" in Query Report - Cumulative tab
Clinical Queries - Queries under "Clinical Review" in Query Report - Cumulative tab
Medical Queries - Queries under "Medical Review" in Query Report - Cumulative tab
Site Queries - count of queries in Query Report - Site Action or Queries under "Site Review" in Query Report - Cumulative tab
Field Monitor Queries - count of queries in Query Report - CRA Action or Queries under "Field Monitor Review" in Query Report - Cumulative tab
Coding Queries - count of uncoded terms 
Safety Queries - Queries under "Safety Review" in Query Report - Cumulative tab

CRFs Require Verification (SDV) - count CRFs in SDV tab having Verification Status as "Require Verification"
Forms Verified - count CRFs in SDV tab having Verification Status as "Verifed"
CRFs Frozen - count of CRFs in CRF Freeze tab
CRFs Not Frozen - count of CRFs in CRF UnFreeze tab
CRFs Locked - count of CRFs in CRF Locked tab
CRFs Unlocked - count of CRFs in CRF UnLocked tab

PDs Confirmed - count of PDs having PD Status as PD Confirmed in Protocol Deviation tab
PDs Proposed - count of PDs having PD Status as PD proposed in Protocol Deviation tab

CRFs Signed - 
CRFs overdue for signs within 45 days of Data entry	- count of CRFs having "No. of days" less than 45
CRFs overdue for signs between 45 to 90 days of Data entry - count of CRFs having "No. of days" between 45 and 90
CRFs overdue for signs beyond 90 days of Data entry	- count of CRFs having "No. of days" more than 90
Broken Signatures - count of CRFs having "Page Require Signature" value as "Yes - Broken Signature"
CRFs Never Signed - count of CRFs having "Page Require Signature" value as "Yes - Never Signed"
